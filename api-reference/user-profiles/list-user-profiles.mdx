title: "List user profiles"
api:
  "GET
  https://api.rownd.io/applications/{app}/users/data?fields=string&lookup_filter=string&id_filter=string"
---

### Query Parameters

<ParamField query="fields" type="string" />

<ParamField query="lookup_filter" type="string">
  The values of verification fields to filter by (e.g., email, phone number).
  Depends on what verification method is used by the underlying application.
  This is a comma-separated list of values.
</ParamField>

<ParamField query="id_filter" type="string">
  A comma-separated list of user IDs to filter by. If any ID is not found, it
  will be excluded from the response.
</ParamField>
