title: "Mobile SDK Overview"
---

Rownd builds the core of its mobile app SDKs using native code, which provides the best performance across all platforms. Whether you're using pure-native or leveraging a multiplatform framework like React Native or Flutter, Rownd has you covered.

Rownd's SDKs have dozens of visual elements that can save you days of nit-picky frontend development and design. 

## Steps for getting started:
1. [Set up your Rownd account](/welcome/getting-started) (only takes a minute).
2. Create an [app key](/configuration/app-credentials).
3. Install Rownd using an SDK.
4. [Configure customizations](/configuration/overview) and sign-in methods from the Rownd platform.
