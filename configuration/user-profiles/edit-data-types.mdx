#### Edit Data Type

1. Go to the Overview page

2. Click on the tab labeled "Data types"

![](/images/Screen20Shot202022-07-2220at204.50.1720PM.png)

Applications page with the Data Types Tab selected

3. Go to the right side of the data type table. Click the vertical menu icon for
   the data type you want to edit.

4. Fill out the Form information and click "Save edits"

![](/images/Screen20Shot202022-07-2220at205.04.4220PM.png)

Edit Data Types modal is open
