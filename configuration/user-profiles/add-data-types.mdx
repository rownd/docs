#### Add Data Type

1. Go to the Overview Page

2. Click on the Tab labeled "Data Types"

![](/images/Screen20Shot202022-07-2220at204.50.1720PM.png)

Applications page with the Data Types Tab selected

3. Click on the "Add data type" button on the right side

4. Fill out the Form information and click "Add"

![](/images/Screen20Shot202022-07-2220at204.52.4020PM.png)

Add Data Types Modal is open
