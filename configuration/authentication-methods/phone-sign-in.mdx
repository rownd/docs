#### When to use phone sign-in.

There are few more powerful ways to bridge the laptop-mobile gap than with Phone Number sign-in. Rownd utilizes powerful sign-up links that make it easy to verify a phone number without having to copy down annoying 4-8 digit passcodes. 

#### Navigating to the Phone number Authentication method

Toggle Phone number to "Allow for Authentication." 

![](/images/image.png)

Note: There is an incremental cost to send SMS text messages. Most Rownd plans include a set number of messages monthly and a small surcharge for any going over monthly allocation.
