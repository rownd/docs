There are two properties that Rownd looks for in the response body that drive behavior when thhe API call completes.

<Expandable title="Response Properties" defaultOpen="true">
  <ResponseField name="return_to">
  A relative path or URL to which Rownd will redirect
  </ResponseField>
  <ResponseField name="should_refresh_page">
  If `true`, Rownd will refresh the current page
  </ResponseField>
</Expandable>
