---
title: "Update a group invite"
openapi: "PUT /me/groups/{group}/invites/{invite}"
---

Update a group invite (user-scoped).

### Endpoint

- Method: `PUT`
- Path: `/me/groups/{group}/invites/{invite}`
- Base URL: `https://api.rownd.io`

### Headers

- `Authorization: Bearer <user-token>`

### Request body

```json
{
  "status": "accepted"
}
```

### Try it

```bash
curl --request PUT \
  --url "https://api.rownd.io/me/groups/{group}/invites/{invite}" \
  --header "Authorization: Bearer <user-token>" \
  --header "Content-Type: application/json" \
  --data '{"status":"accepted"}'
```

### Response

200 - application/json

```json
{
  "id": "inv_abc",
  "status": "accepted"
}
```
