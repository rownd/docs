Retrieve the current JWK set that validates Rownd-issued tokens.

<Endpoint method="GET" path="/hub/auth/keys" baseUrls={['https://api.rownd.io','https://api.us-east-2.dev.rownd.io']} />

### Try it

```bash title="cURL"
curl --request GET \
  --url https://api.rownd.io/hub/auth/keys
```
